Сервис форвардинга сетевых портов, для возможности подключения сетевых устройств не имеющих белого IP адреса.

SSL режим

Клиент                          Сервер
  |                              |
  |-------- SSL Connect -------->|
  |                              |
  | Проверяет сертификат сервера |
  |                              | Принимает SSL соединение
  |                              | (не требует сертификат клиента)
  |<----- SSL Established -------|
  |                              |
  |====== Secure Channel ========|
